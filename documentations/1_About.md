# About Project
本项目实现面向顾客的扫码点餐系统和面向商家的后台管理系统。扫码点餐系统支持用户到餐厅扫描餐桌二维码进入特定餐厅应用，获得点菜、结算等服务。后台管理系统支持商家对餐厅应用上菜式的增删改查，以及对顾客订单的管理。
